# Prerequisite
  - Microsoft Face Recognition API
  - Camera
  - Public IP
  - Tomcat with https

# Steps
  - Change Authentication.java
  - Change config.properties and log4j2.xml
  - Build and deploy to tomcat

# Demo
  - Start up Tomcat and visit https://[hostname]/faceRecognition/
